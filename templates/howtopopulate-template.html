populating parcels data into elasticsearch

a python script is developed to normalize the input addresses. This script could be viewed here . The script expects the addresses to be relatively cleaned and the parcel id's assigned to each row.

two python modules: address_normalizer and objData should be located in working directory. If using the jupyter notebook in the above link, it's located in Jupyter's home directory.

the task of the script is to read the input addresses, and after normalizing them, to make a call to the document service then importing to elasticsearch.
